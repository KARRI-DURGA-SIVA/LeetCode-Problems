# LeetCode Problems

Java solutions organized topic-wise and difficulty-wise for clean revision.

## Folder Structure

```text
LeetCode-Problems/
├── Basic-java-methods/
│   └── Methods.java
├── Arrays/
│   ├── Easy/
│   │   └── ShortestCompletingWord.java
│   ├── Medium/
│   │   └── NumberItSelf.java
│   └── Hard/
└── Strings/
    ├── Easy/
    ├── Medium/
    │   └── Longestpalindrome.java
    └── Hard/
```

## Problems

| Topic | Difficulty | Problem | File |
| --- | --- | --- | --- |
| Basic Java Methods | Practice | String methods demo | `Basic-java-methods/Methods.java` |
| Arrays | Easy | Shortest Completing Word | `Arrays/Easy/ShortestCompletingWord.java` |
| Arrays | Medium | Product of Array Except Self | `Arrays/Medium/NumberItSelf.java` |
| Strings | Medium | Longest Palindromic Substring | `Strings/Medium/Longestpalindrome.java` |

## How to Run

Compile from the repository root:

```bash
javac Strings/Medium/Longestpalindrome.java
```

Run files that contain a `main` method:

```bash
javac Basic-java-methods/Methods.java
java -cp Basic-java-methods Methods
```

## Notes

- Add each new solution inside its topic folder.
- Use `Easy`, `Medium`, and `Hard` folders for difficulty.
- Keep one problem solution per file.
