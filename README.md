# Java Practice Repository

This repository contains Java practice programs organized by topic. The structure is kept simple for GitHub and daily revision: basic Java methods, array problems, and string problems.

## Folder Structure

```text
.
├── Basic-java-methods/
├── Arrays/
│   ├── Easy/
│   ├── Medium/
│   └── Hard/
└── Strings/
    ├── Easy/
    ├── Medium/
    └── Hard/
```

## Topic-Wise Problems

| Topic | Difficulty | Problem | File |
| --- | --- | --- | --- |
| Basic Java Methods | Practice | String methods demo | `Basic-java-methods/Methods.java` |
| Arrays | Easy | Shortest Completing Word | `Arrays/Easy/ShortestCompletingWord.java` |
| Arrays | Medium | Product of Array Except Self | `Arrays/Medium/NumberItSelf.java` |
| Strings | Medium | Longest Palindromic Substring | `Strings/Medium/Longestpalindrome.java` |

## How to Run

Compile a Java file from the repository root:

```bash
javac Arrays/Easy/ShortestCompletingWord.java
```

Run a file that has a `main` method:

```bash
javac Basic-java-methods/Methods.java
java -cp Basic-java-methods Methods
```

## Notes

- Add new problems inside the correct topic and difficulty folder.
- Use clear file names that match the problem name.
- Keep each file focused on one problem or concept.
